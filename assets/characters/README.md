# Character Image Assets

Add licensed or approved recognizable character images here. The app does not use stylized stand-ins for this section.

The Fantasy/Sci-Fi Character Match result page loads these files automatically when they exist:

- `gandalf.jpg`
- `hermione-granger.jpg`
- `luke-skywalker.jpg`
- `the-doctor.jpg`

If an image is missing, the result page tries a recognizable Wikipedia/Wikimedia thumbnail. Local approved files still take priority when added here.
